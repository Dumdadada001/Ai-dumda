# 📋 작업 브리프

**원 명령:** 우리 유투브 사이트를 내화면에 띄울수 있나요? 가능하면 진행해봐주세요

## 요약
회사 내부 대시보드 또는 시스템 화면에 유튜브 채널의 콘텐츠(영상 플레이어 혹은 피드)를 임베딩하는 기술 구현 요청입니다.
이를 위해 기술적 타당성을 검토하고, 개발자가 실제 구동할 수 있는 자동화된 계획을 수립해야 합니다.

## 분배
- **🔍 Researcher**: 회사 내부 시스템에 유튜브 콘텐츠를 임베딩하는 기술적 가능성(iframe 사용의 적절성, API 제한 사항 등)을 조사하고, 가장 안정적이고 트래픽 부하가 적은 최적의 임베드 방식을 보고해주세요. (예: YouTube Data API v3 활용 여부)
- **💻 Developer**: 연구 결과를 바탕으로, 내부 화면에 유튜브 플레이어 및 최신 영상 피드를 자동으로 불러오는 시스템 통합(Integration) 계획을 수립하고 필요한 코드 구조 설계 초안을 작성해주세요. (프론트엔드/백엔드 분리 명시)
