# ✍️ Writer — 위에서 확정된 숏폼 콘텐츠 기둥과 구조에 맞춰, 시청자의 스크롤을 멈추게 할 수 있는 강력한 '후크(Hook) 문구' 10개와 초기 스크립트 초안 3개를 작성합니다.

✍️ Writer: 작업 시작합니다. CEO님의 지시에 따라, 앞서 확정된 'Pain Point 기반 솔루션 제공' 구조에 최적화된 후크 문구 10개와 즉시 활용 가능한 스크립트 초안 3개를 작성했습니다.

이 산출물은 단지 아이디어가 아니라, **재료가 준비된 레고 블록**입니다. 각 스크립트는 녹음만 하면 바로 영상으로 만들 수 있도록 대본화했으며, 시청자가 가장 많이 이탈하는 0~2초 구간에 모든 역량을 집중했습니다.

---

### 🔥 파트 1: 스크롤을 멈추는 강력한 후크 문구 10가지 (Hook Library)

후크는 텍스트 자막으로 가장 크고 충격적으로 노출되어야 하며, 내레이션과 함께 사용됩니다. 각 후크 옆에 어떤 주제군(Pillar)에 적합한지 명시했습니다.

**✅ [AI/Tech Utility] (정보의 격차를 이용)**
1. "혹시 아직도 AI에게 이렇게 질문하시나요? 90%는 실패합니다." (최대 고효율 프롬프트 비법 예고)
2. "이 기능 하나 모르면, 업무 자동화에 돈을 날리는 겁니다." (특정 소프트웨어의 숨겨진 고급 기능 공개)
3. "챗GPT가 답변할 때 절대 이렇게 하지 마세요. 답이 이상해지는 이유입니다." (AI 사용 시 흔한 실수 지적)

**✅ [마이크로 재테크/생산성 해킹] (공포 및 절약 욕구 자극)**
4. "매달 5만 원씩 새고 있는 돈, 당신은 지금 자동이체에 속고 있습니다." (생활 밀착형 소비 패턴 지적)
5. "남들은 아는 '시간 단축 루틴' 딱 3가지만 알려드릴게요. 오늘부터 적용하세요." (즉각적인 행동 변화 유도)
6. "월급 빼고 다 오르는 시대, 돈 버는 법은 이 원리 하나로 돌아갑니다." (경제적 원리를 간단하게 설명하며 기대감 증폭)

**✅ [커뮤니티 Q&A/Listicle] (질문-답변 구조 활용)**
7. "직장인들이 가장 많이 묻지만, 아무도 알려주지 않는 OOO의 비밀입니다." (금기시되거나 숨겨진 정보를 건드림)
8. "만약 [특정 상황]을 피하고 싶다면? 오늘 딱 이 것만 기억하세요." (위험 회피 심리를 자극하는 간결한 팁)
9. "이거 하나로 끝납니다. '복잡하다'는 생각을 지워주는 만능 솔루션." (과도하게 복잡한 문제를 간단히 해결할 수 있음을 약속)

**✅ [밈/트렌드 포착] (공감과 충격 결합)**
10. "요즘 MZ들이 열광하는 '그 감성', 사실은 이것만 조합하면 만들어집니다." (최신 트렌드의 본질적 원리를 파헤치며 흥미 유발)

### 🎬 파트 2: 스크립트 초안 3가지 (실행 가능한 대본)

모든 스크립트는 **[시간] - [내레이션/대사] - [화면 지시]**의 구조를 따르며, 녹음 후 바로 편집에 들어갈 수 있도록 구체적인 비주얼 가이드라인을 포함했습니다.

#### 📝 Script Draft 1: AI 프롬프트 마스터 (AI/Tech Utility)
*   **목표:** Save 유도 (나중에 써보고 싶게 만들기)
*   **길이 예측:** 25~30초
*   **주제:** '역할 부여(Role-Playing)'와 '출력 형식 지정'을 통한 프롬프트 고도화

| 시간 | 요소 | 스크립트 (Tone: 단호함, 자신감) | 화면 지시 및 편집 가이드 |
| :---: | :--- | :--- | :--- |
| **0~2초** | **[HOOK] Pain Point** | "🚨 잠깐 멈추세요. 당신이 AI에게 질문하는 방식으로는 절대로 완벽한 답을 얻을 수 없습니다." (강조하며 화면 가리키기) | *자막:* '❌이렇게 물어보면 안 돼요!' / 충격적인 BGM 시작. / 배경은 흐릿하고 불안정한 느낌. |
| **2~10초** | **[BRIDGE] 문제 정의** | "왜 그럴까요? AI는 그냥 정보 검색기가 아니기 때문입니다. 우리가 해야 할 건, AI에게 '역할'을 부여하는 겁니다." | *비주얼:* 뇌 속 전구가 켜지는 애니메이션 (깨달음). / 키워드: **[역할 지정]**, **[규칙 설정]**를 크고 명확하게 자막 처리. |
| **10~25초** | **[BODY] 솔루션 제시** | "딱 두 가지만 추가하세요. 첫째, '당신은 지금부터 베테랑 마케터입니다.' 둘째, '답변은 반드시 표(Table) 형식으로 정리해주세요.' 이 두 문장만 붙이면, 답변의 질이 달라집니다." | *화면 녹화:* 실제 챗GPT 화면을 보여주며 (Before/After 비교). / **'Role:'** 와 **'Format:'** 키워드를 하이라이트 처리. 단계별 리스트 애니메이션 활용. |
| **25~30초** | **[CTA] 다음 액션** | "이 팁, 저장 안 하면 나중에 잊어버립니다. 지금 바로 '저장' 버튼 눌러두세요!" | *자막:* ⭐Save 필수!⭐ / 시청자가 화면의 '저장(Save)' 버튼을 누르는 동작을 손가락으로 유도. |

#### 📝 Script Draft 2: 월급 순삭 방지법 (Micro Finance/Productivity)
*   **목표:** Comment 유도 (댓글로 공감 및 질문 받기)
*   **길이 예측:** 30~40초
*   **주제:** '무심코 지나치는 구독 서비스' 점검 리스트

| 시간 | 요소 | 스크립트 (Tone: 친근함, 걱정하는 듯한 공감) | 화면 지시 및 편집 가이드 |
| :---: | :--- | :--- | :--- |
| **0~2초** | **[HOOK] Pain Point** | "🚨혹시 매달 통장에서 나가는 돈 때문에 불안하신 적 있으신가요? 지금 당장 확인하세요." (궁금증 유발) | *자막:* '💸이거 모르면 손해예요!' / 배경은 지갑이나 은행 앱 화면의 흐릿한 모습. / BGM: 약간 긴박하면서도 친근한 느낌. |
| **2~15초** | **[BRIDGE] 문제 정의** | "우리가 가장 많이 놓치는 돈은, '자동 결제'되는 서비스들입니다. 넷플릭스? 운동 앱? 커피 쿠폰 적립 같은 것까지 다요." (구체적인 예시 언급) | *비주얼:* 다양한 구독 서비스 로고가 빠르게 지나감. / 핵심 원리: **[지출의 가치 검토]**를 자막으로 강조하며 전문성을 확보. |
| **15~35초** | **[BODY] 솔루션 제시** | "딱 3분만 투자하세요. ① 명세서를 다운받아 '사용 빈도'가 낮은 것부터 지우고, ② 가족들과 함께 공유하여 누가 쓸지 정하고, ③ 대신 가성비 좋은 무료 대안이 있는지 검색해보세요." (구체적인 행동 지침) | *템플릿:* 체크리스트 애니메이션 사용. ①②③ 번호가 순차적으로 나타남. / 배경은 깔끔한 노트와 펜 이미지 등 '정돈'의 느낌을 주는 비주얼로 전환. |
| **35~40초** | **[CTA] 다음 액션** | "여러분은 어떤 자동 결제에 속고 계신가요? 댓글로 여러분만의 절약 노하우를 알려주세요!" | *자막:* 💬댓글 고고! (커뮤니티 참여 유도) / 시청자가 손가락으로 '댓글' 영역을 가리키는 액션. |

#### 📝 Script Draft 3: 생산성 해킹 리스트 (Listicle/General Utility)
*   **목표:** Share 및 High Retention 유도 (나에게 필요한 정보로 인식시키기)
*   **길이 예측:** 15~20초 (가장 짧고 임팩트 있게)
*   **주제:** 업무 효율을 극대화하는 필수 단축키/습관 TOP 3

| 시간 | 요소 | 스크립트 (Tone: 빠름, 명쾌함) | 화면 지시 및 편집 가이드 |
| :---: | :--- | :--- | :--- |
| **0~2초** | **[HOOK] Pain Point** | "🚨아직도 마우스로 클릭하고 계신가요? 이 3개만 알면 당신은 업무 속도가 두 배 빨라집니다." (빠르고 단호하게) | *자막:* '🖱️마우스는 버리세요!' / 긴장감 있는 짧고 날카로운 사운드 효과. |
| **2~18초** | **[BODY] 솔루션 제시** | "첫째, [Ctrl+Shift+T]. 창이 닫혀도 이전 페이지로 돌아가는 마법의 단축키입니다. 둘째, [Command+F]. 검색창을 활용해 문서 전체를 순식간에 찾으세요. 마지막으로, '오늘 할 일'은 아침에 3가지만 적어보세요." (핵심만 빠르게 리스트업) | *비주얼:* 실제 키보드 위 단축키 조합이 번쩍이며 애니메이션 효과와 함께 등장. / 각 항목당 배경 색상을 다르게 하여 명확히 구분. / 빠르고 경쾌한 편집 속도 유지. |
| **18~20초** | **[CTA] 다음 액션** | "가장 유용했던 팁은 무엇인가요? 친구에게 공유할 만한 정보가 있다면 지금 바로 보내세요!" | *자막:* 📤친구 태그 필수! / 화면의 '공유' 버튼을 손으로 가리키며 시각적 행동 지시. |
